✓ Requirement #13: Block Elements

Block elements take up the full width available and start on a new line.

<div>, <p>, <h1-h6>, <header>, <main>, <footer>, <section>

Examples: This paragraph, the div containers, and all heading elements are block-level.

✓ Requirement #14: Inline Elements

Inline elements only take up as much width as necessary and don't start on a new line. For example: this strong tag, this emphasis tag, this link, and this span element are all inline elements that flow within the text.

✓ Requirement #15: Inline-Block Elements

Inline-block elements flow inline but can have width and height like blocks:

Box 1 Box 2 Box 3

These boxes flow inline but respect padding, margin, width, and height.

✓ Requirement #18: ID Selector

This section has id="id-selector-demo" and the #hero-section above has unique styling.

CSS: #hero-section { background: gradient; padding: 60px; }
CSS: #top { scroll-margin-top: 96px; }

ID selectors are unique and styled with # in CSS.
